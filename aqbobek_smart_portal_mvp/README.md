🏫 Aqbobek Smart Portal


Run: pip install -r requirements.txt python app.py



Aqbobek Smart Portal - это единая школьная платформа, которая объединяет учеников и учителей в одном цифровом пространстве. Платформа упрощает обучение, ускоряет обратную связь и делает работу с успеваемостью более наглядной и удобной.

📌 Цели проекта
Объединить ученика и учителя в одном цифровом пространстве
Упростить работу с оценками и аналитикой успеваемости
Повысить вовлечённость учеников через геймификацию
Снизить нагрузку на учителя за счёт автоматизации анализа данных

👩‍🎓 Функции для учеников
Просмотр оценок, предметов и среднего балла
Личный кабинет с наглядной статистикой
Система XP - элемент геймификации для мотивации
AI-анализ: рекомендации по улучшению знаний
Получение уведомлений о новых заданиях и оценках

👨‍🏫 Функции для учителей
Панель управления классом: средний балл, динамика успеваемости
Определение учеников в зоне риска
Просмотр слабых тем класса
Доступ к профилям учеников, оценкам и XP
Возможность назначать дополнительные уроки и выставлять оценки с мгновенными уведомлениями

🤖 AI функции
Анализ успеваемости учеников и класса в целом
Рекомендации по слабым темам
Генерация домашнего задания для всей группы при выявлении общих проблем

🖥️ Kiosk режим
Позволяет открывать портал на электронной доске учителя
Демонстрация заданий, аналитики и информации для всего класса
Удобен для наглядных уроков без входа в личный кабинет

🎮 Геймификация
XP система мотивирует учеников
Баллы начисляются за активность, выполнение заданий и успехи
Позволяет отслеживать прогресс и стимулирует развитие

⚡ Технологии
Frontend: HTML

Backend: Python

AI-модуль: базовые рекомендации и подсказки

Дополнительно: Kiosk-режим для электронной доски
